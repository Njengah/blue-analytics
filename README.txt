Simple WordPress plugin to add Google Analytics. 

== Description ==

Plugin provides user with a backend textarea where they can copy the GA code and choose to enable or disable the anonymous option. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `blue-analytics.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= How do you set up Blue Analytics =

After installation, check the General option menu for the Blue Analytics submenu. Click on the submenu and you will be redirected to the settings page where you can add the GA Code.
